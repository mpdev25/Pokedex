# Pokedex
A simple pokedexcli to explore locations in Pokeapi, discover which Pokemon are available and catch tham to add to your pokedex. 
Use map to return a list of twenty locations. Keep using map to see more locations.
Use mapb to go back to the previous twenty locations.
Use explore followed by the name of a location to see which Pokemon are there.
Use catch with the name of a Pokemon to try to catch it.
Use inspect with the name of a Pokemon to see it's statistics.
use pokedex to see your Pokemon collection.
Use exit to terminate the program.
